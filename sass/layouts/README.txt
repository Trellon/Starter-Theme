Layouts should be kept in this folder.

This is a SMACSS theme. Layout rules are discussed here, and you should understand them.

https://smacss.com/book/type-layout

Some guidelines to follow.

1. Use mixins to define layouts. We want to be able to apply them over and over again 
   without forcing someone to add a class in the markup. Create mixins and apply them 
   in the theme file.
